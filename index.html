Dependencies

You'll need:

  1. JQuery (this dependency should be removed eventually)
  2. these bitmap fonts. Put them in your www directory.

Download

All you need to get started is Pxxl.js and these bitmap fonts.

Or you can download the whole project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/remcoder/Pxxl.js
