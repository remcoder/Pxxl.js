HTML5 Canvas utility to render texts in retro style and more.

Get Started

  1. include the pxxl-0.1.min.js script.
  2. include JQuery (this dependency should be removed soon)
  3. put these bitmap fonts in your www directory

To get started immediately, create a canvas element and just call the 'PixelText' function: 


      <canvas />
      <script>
      PixelText($("canvas") , { 
        text: 'hello world!',
        font : 'fonts/c64.bdf',
        style : 'squares',
        scale: '10',
        fill: "#66f",
        stroke: "black"
      });
      </script>
